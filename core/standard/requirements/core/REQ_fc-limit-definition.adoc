|*Requirement {counter:req-id}* |/req/core/fc-limit-definition +

Each feature collection operation SHALL support a parameter `limit`
with the following characteristics (using an OpenAPI Specification 3.0 fragment):

[source,YAML]
----
name: limit
in: query
required: false
schema:
  type: integer
  minimum: 1
  maximum: 10000
  default: 10
style: form
explode: false
----
|===
