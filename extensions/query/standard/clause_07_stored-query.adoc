[[clause-stored-query]]
== Requirements Class "Stored Query"

=== Overview

include::requirements/requirements_class_stored-query.adoc[]

A stored query is a predefined query that is available as a resource in a Web API.  This clause only deals with un-parameterized stored queries.  The <<clause-parameterized-stored-query,Parameterized Stored Query>> clause deals with stored queries that take parameters.
