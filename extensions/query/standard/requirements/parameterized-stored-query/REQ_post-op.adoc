[[req_parameterized-stored-query_post-op]]
[width="90%",cols="2,6a"]
|===
^|*Requirement {counter:req-id}* |*/req/parameterized-stored-query/post-op*
^|A |For every stored query identified in the stored queries response (path `/query`), the server SHALL support the HTTP POST operation at the path `/query/{queryId}`.
^|B |The parameter `queryId` is each `id` property in the stored queries response (JSONPath: `$.queries[*].id`).
^|C |The content type of the POST body shall be `application/x-www-form-urlencoded`.
|===
