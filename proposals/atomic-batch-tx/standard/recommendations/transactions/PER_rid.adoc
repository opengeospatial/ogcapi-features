|===
^|*Permission {counter:per-id}* |*/per/transactions/rid*
^|A |If the representation of the resource within an insert action within a transaction contains a resource identifier, the server MAY use this identifier as the new resource identifier in the collection or the server MAY ignore the value and assign its own identifier for the resource.
