[[sc_async-transactions]]
== Requirements Class "Asynchronous Transactions"

=== Overview

include::requirements/requirements_class_async-transactions.adoc[]

Because a transaction may operation of a large number of resources spanning one or more collections it is entily possible that processing the transaction may take more time than the HTTP timeout interval.  

In order to accomodate such long-running operations, this requirements class defines the requirement for executing transactions asynchronously, for monitoring the execution of such transactions and for retrieving the results of processing.
