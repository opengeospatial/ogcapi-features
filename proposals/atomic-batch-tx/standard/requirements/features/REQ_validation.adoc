[[req_features_validation]]
[width="90%",cols="2,6a"]
|===
^|*Requirement {counter:req-id}* |*/req/features/validation*
^|Condition |Server implements <<OAFeat-1,OGC API - Features - Part 1: Core, Requirements Class "GeoJSON">>
^|Condition |Server implements <<OAFeat-5,OGC API - Features - Part 5: Schemas, Requirements Class "Returnables and Receivables">>
^|A |The server SHALL support the HTTP GET method at `/collections/{collectionId}/schema`.
^|B |`200`-responses of the server to these requests SHALL support the media type `application/schema+json`.
^|C |The response content SHALL be a JSON Schema that a feature encoded in a transaction has to validate against.
|===
